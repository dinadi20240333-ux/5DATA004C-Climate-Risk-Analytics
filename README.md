# 5DATA004C-Climate-Risk-Analytics
Data Science Project Lifecycle Coursework – Climate Risk Analytics for Aethelgard Re
This repository contains the code, datasets, outputs, and supporting files developed for the **5DATA004C Data Science Project Lifecycle** coursework.

## Project Overview

The aim of this project is to investigate the relationship between global warming factors and global temperature anomaly using historical climate datasets. The project applies the Data Science Project Lifecycle to analyse climate trends, develop predictive regression models, and generate evidence-based recommendations for the fictional reinsurance company **Aethelgard Re**.

## Project Objectives

- Collect and preprocess multiple climate datasets.
- Perform exploratory data analysis (EDA).
- Investigate correlations between climate variables.
- Assess multicollinearity using Variance Inflation Factor (VIF).
- Develop and compare regression models.
- Forecast future global temperature anomalies.
- Provide business recommendations for climate risk management.
